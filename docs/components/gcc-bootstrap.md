# GCC Bootstrap Documentation

## Overview

GCC Bootstrap compiler (Stage 2) - minimal C-only compiler for building glibc.

## Build Information

- **Build Script**: `scripts/build-gcc-bootstrap.sh`
- **Purpose**: Build glibc (no C++ support)
